Overview

The project turns "best practice" into a curated, versioned evidence corpus and evaluates real-world documentation against that bar. Ambiguity is labeled, not hidden.

Aligns data streams

Builds longitudinal patient trajectories by linking notes and device snapshots across visits.

Flags measurable gaps

Starts with one rigorously defined gap, then expands via a shared taxonomy.

Explains each flag

Every output includes evidence grounding, uncertainty tags, and auditability.

Gap Definition (v1)

Persistent clinical seizures across two visits with no evidence-aligned optimization attempt.

Trigger & window

  • Clinical seizures documented at ≥2 consecutive visits
  • PMDS used to corroborate objective burden and parameter-change evidence

Expected response

  • Optimization attempt or actionable programming plan
  • Escalation action class or documented rationale to defer
